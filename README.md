# ぷにぷにカート3D (Puni Puni Kart 3D) 🏎️💨✨

温かみのあるおもちゃの世界（ビスケットの道路やキャラメルの縁取り）を走り抜ける、Three.js を使ったポップで本格的な **3D WebGL カートレースゲーム** です。

👉 **[ゲームを今すぐ遊ぶ！ (Live Demo)](https://wataameto.github.io/game3/)**

---

## 🎮 操作方法 (Controls)

| キーボード入力 | アクション | Description |
| :--- | :--- | :--- |
| **`↑` / `W`** | **アクセル (Forward)** | カートを前進・加速させます。 |
| **`↓` / `S`** | **ブレーキ/バック (Backward)** | カートを減速・後退させます。 |
| **`←` / `A`** | **左ハンドル (Turn Left)** | カートを左に旋回させます。 |
| **`→` / `D`** | **右ハンドル (Turn Right)** | カートを右に旋回させます。 |
| **`Space`** | **ドリフト (Drift)** | 旋回中に長押しすると、お尻を滑らせるドリフトが決まります！ |

💡 **ドリフトのコツ**:
スピードに乗った状態で左右どちらかにハンドルを切りながら `Space` キーを長押しすると、車体を内側に傾けながら滑らかに横滑り（ドリフト）します。ドリフト中はタイヤから可愛いピンクの煙が噴き出します！

## 🌟 ゲームの特徴 (Features)

### 1. 🤖 3台の対戦ライバル（ラバーバンドAI）
自動でコースを周回する3台の個性豊かなライバルカートが参戦します：
- **メロンカート 🍏** (グリーン)
- **プリンカート 🍮** (イエロー)
- **ぶどうカート 🍇** (パープル)

ライバルたちはプレイヤーとの距離を常に監視しており、プレイヤーがリードすると本気を出して追い上げ、プレイヤーが遅れると手加減して待ってくれる **「ラバーバンドAI」** を搭載しています。常にデッドヒートの緊張感が楽しめます。

### 2. 💥 追突クラッシュ ＆ スピン失速システム
後ろから前のカートに追突（アタック）すると、**ぶつかられた車が一時的なクラッシュ（スピン ＆ 失速）状態** になります！
- クラッシュした車はクルクルと激しく回転し、車体がガタガタ歪み、マフラーからココア色の黒煙を吹き出します。
- 約0.6秒間は速度が極端に低下（ちょっと止まる）するため、相手を出し抜く大チャンスになります。
- プレイヤー対ライバルだけでなく、**ライバル同士もぶつかり合ってスピンし合う** ため、レース展開が目まぐるしく変化します。

### 3. 🛡️ スマートハンドルアシスト ＆ 壁スライド物理
3Dレースゲームに不慣れな方でも快適にドライブできるよう、2つのアシストを常時搭載しています：
- **ハンドルアシスト**: コースの端に近づくと、自動的に道路 of カーブに沿うように車の向きを自動で微調整します。
- **壁スライドクランプ**: コース外への落下を防ぐ見えないガードレール物理です。端にぶつかっても外へ落ちず、オレンジ色の火花を散らしながら「壁ずり」でスライド前進し続けます。

### 4. 🪙 コイン ＆ 加速床
コース上に回転する3Dコインと、踏むと急加速するグリーンの加速床（ダッシュ板）が配置されています。

## 🛠️ ローカルでの起動方法 (Local Development)

本ゲームは静的 HTML/JS ファイルで構成されていますが、WebGL リソースのロードや Playwright テストを実行するために、ローカル Web サーバー経由での起動を推奨します。

### 1. 依存関係のインストール (オプション)
Playwright によるテストを実行する場合のみ必要です。
```bash
npm install
```

### 2. ローカルサーバーの起動
`http-server` などの静的サーバーを使って起動します。
```bash
# http-server を使ってポート 8083 で起動する場合
npx http-server -p 8083
```

### 3. ブラウザでアクセス
ブラウザで以下のURLを開いてプレイします：
👉 **[http://127.0.0.1:8083/index.html](http://127.0.0.1:8083/index.html)**
